Solution 3.4.16.1.

For simplicity, we will suppose \(X\) and \(Y\) are not in percents but are in decimal form (e.g. if Caterpillar’s stock increases 1%, then \(X = 0.01\text{;}\) or if it loses 1%, then \(X = -0.01\)). Then we can write an equation for Leonard’s gain as
\begin{equation*} \$6000 \times X + \$2000 \times Y \end{equation*}
If we plug in the change in the stock value for \(X\) and \(Y\text{,}\) this equation gives the change in value of Leonard’s stock portfolio for the month. A positive value represents a gain, and a negative value represents a loss.
