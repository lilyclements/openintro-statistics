(a) First find \(P(6)=5/36\text{,}\) then use the complement: \(P(\text{not } 6) = 1 - P(6) = 31/36\text{.}\) (b) First find the complement, which requires much less effort: \(P(2 \text{ or } 3)=1/36+2/36=1/12\text{.}\) Then calculate \(P(B) = 1-P(B^c) = 1-1/12 = 11/12\text{.}\) (c) As before, finding the complement is the clever way to determine \(P(D)\text{.}\) First find \(P(D^c) = P(11 \text{ or } 12)=2/36 + 1/36=1/12\text{.}\) Then calculate \(P(D) = 1 - P(D^c) = 11/12\text{.}\)
