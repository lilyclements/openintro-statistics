Solution 4.4.9.1.

In each part, \(p=0.7\text{.}\)
  1. The number of days is fixed, so this is binomial. The parameters are \(k=3\) and \(n=7\text{:}\) 0.097.
  2. The last “success” (admitting a heart attack patient) is fixed to the last day, so we should apply the negative binomial distribution. The parameters are \(k=2\text{,}\) \(n=4\text{:}\) 0.132.
  3. This problem is negative binomial with \(k=1\) and \(n=5\text{:}\) 0.006. Note that the negative binomial case when \(k=1\) is the same as using the geometric distribution.
